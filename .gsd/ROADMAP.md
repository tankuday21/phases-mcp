# ROADMAP.md

> **Current Phase**: Not started
> **Milestone**: v1.0

## Must-Haves (from SPEC)
- [ ] Users can create, edit, and delete markdown notes
- [ ] Live markdown preview renders as user types
- [ ] Notes persist across browser sessions via localStorage
- [ ] Search filters notes by title/content in real-time
- [ ] Tags can be added/removed and used to filter notes
- [ ] UI is responsive, dark-themed, and visually premium

## Phases

### Phase 1: Core UI & Editor
**Status**: ✅ Complete
**Objective**: Build the HTML structure, CSS design system, and split-pane markdown editor with live preview

### Phase 2: Note CRUD & Persistence
**Status**: ⬜ Not Started
**Objective**: Implement create, read, update, delete notes with localStorage persistence and note list sidebar

### Phase 3: Search, Tags & Polish
**Status**: ⬜ Not Started
**Objective**: Add real-time search, tagging system, responsive polish, and micro-animations


### Phase 4: Notion-Style Floating Toolbar
**Status**: ✅ Complete
**Objective**: Add a floating formatting toolbar that appears when text is selected in the editor, with options for Bold, Italic, Strikethrough, Code, Heading, Link, Quote, and List formatting
