View on GitHub

1 Install Globally via npm

Install the Phases MCP Server package globally onto your machine.

Terminal
# Using npm
npm install -g phases-mcp-server

2 Configure MCP Client

Add Phases to your MCP client configuration (e.g., Claude Desktop, Cursor, Antigravity).

mcp_config.json
{
  "mcpServers": {
    "phases": {
      "command": "phases-mcp-server",
      "transportType": "stdio"
    }
  }
}
